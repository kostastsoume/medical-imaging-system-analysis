Εισαγωγή¶

Δίνεται η ακόλουθη διάταξη που αποτελείται από παραλληλεπίπεδο με γραμμικό συντελεστή απορρόφησης μ0=0.1cm-1. Στο σημείο Α (96,128,64) βρίσκεται το κέντρο ομογενούς κύβoυ με μ1=0.15cm-1 και πλευρά R1=15mm και στο σημείο Β (180,128,64) βρίσκεται το κέντρο ομογενούς κύβoυ με μ2 =0.05cm-1 και πλευρά R2=25mm. Η διάταξη ακτινοβολείται με ακτίνες Χ (X-rays) όπως φαίνεται στο σχήμα.

Θεωρείστε μέγεθος pixel 1mm × 1mm και κατασκευάστε την παραγόμενη ακτινογραφία, στις ακόλουθες περιπτώσεις:

  • Α) σε κάθε mm2

, τα προσπίπτοντα φωτόνια X είναι 1000,

  • Β) σε κάθε mm2

, τα προσπίπτοντα φωτόνια X είναι 100.

Επαναλάβετε την κατασκευή της ραδιογραφικής εικόνας θεωρώντας και τον θόρυβο Poisson.

% Sphere A
m1 = 0.15; % cm^-1
R1 = 25; % mm

% Sphere B
m2 = 0.05; % cm^-1
R2 = 12; % mm

% Parallelogram
m = 0.1; % cm^-1
w = 256; % mm ---> x
h = 256; % mm ---> y
d = 128; % mm ---> z

% Create grids using ndgrid
[X, Y, Z] = ndgrid(1:w, 1:h, 1:d);

% Calculate distances for each sphere (from their center)
distanceA = sqrt((X - 96).^2 + (Y - 128).^2 + (Z - 64).^2);
distanceB = sqrt((X - 180).^2 + (Y - 128).^2 + (Z - 64).^2);

% Create matrix M to store attenuation coefficient.
M = m * ones(w, h, d); % initialize M to the attenuation coefficient of the parallelogram
M(distanceA <= R1) = m1; % if distanceA <= R1 == true : m1 is stored.
M(distanceB <= R2) = m2; % if distanceB <= R2 == true : m2 is stored.
% Else, the values of m don't change

% Case A (1000 photons per mm^2)
IoA = 1000; 
IdA = IoA * exp(-sum(M, 3) * 0.1); % sum along the 3rd dimension (z-axis)
noiseA = -log(rand(size(IdA))) .* IdA; % Alternative to poissrnd

% Normalize ideal image independently
IdA_norm = (IdA - min(IdA(:))) / (max(IdA(:)) - min(IdA(:))); 

% Normalize noisy image with enhancement factor
enhancement_factor = 5; % Boost noisy image visibility
noiseA_norm = (noiseA - min(noiseA(:))) / (max(noiseA(:)) - min(noiseA(:)));
noiseA_norm = noiseA_norm * enhancement_factor; % Enhance brightness
noiseA_norm(noiseA_norm > 1) = 1; % Clip values to [0, 1]

% Display normalized images side by side
figure;
imshow([IdA_norm, noiseA_norm], []); % Display side by side
colormap(gray); % Use gray colormap
title('Case A: 1000 photons per mm^2');
% Save the figure as an image (high resolution)
saveas(gcf, 'caseA_figure_high_res.png');

% Case B (100 photons per mm^2)
IoB = 100; 
IdB = IoB * exp(-sum(M, 3) * 0.1);
noiseB = -log(rand(size(IdB))) .* IdB; % Alternative to poissrnd

% Normalize ideal image independently
IdB_norm = (IdB - min(IdB(:))) / (max(IdB(:)) - min(IdB(:))); 

% Normalize noisy image with enhancement factor
noiseB_norm = (noiseB - min(noiseB(:))) / (max(noiseB(:)) - min(noiseB(:)));
noiseB_norm = noiseB_norm * enhancement_factor; % Enhance brightness
noiseB_norm(noiseB_norm > 1) = 1; % Clip values to [0, 1]

% Display normalized images side by side
figure;
imshow([IdB_norm, noiseB_norm], []); % Display side by side
colormap(gray); % Use gray colormap
title('Case B: 100 photons per mm^2');
% Save the figure as an image (high resolution)
saveas(gcf, 'caseB_figure_high_res.png');
