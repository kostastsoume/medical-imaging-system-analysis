# Medical Imaging System Analysis

University project developed for the **Medical Imaging Systems** course at the **University of Thessaly**.

## Overview

This project focuses on the quantitative evaluation of medical imaging systems and includes:

- Modulation Transfer Function (MTF) estimation
- Gamma camera field homogeneity analysis
- X-ray attenuation simulation
- Image quality assessment
- Comparison of medical imaging system specifications

## Main Results

### Modulation Transfer Function

The MTF analysis was performed using a quadrant phantom containing patterns of different spatial frequencies.

- Maximum intensity: **143**
- Minimum intensity: **35**
- Calculated MTF: **0.6067**

### Gamma Camera Field Homogeneity

Field homogeneity was evaluated using a gamma camera image and a local 13 × 13 pixel analysis window.

- Maximum intensity: **200**
- Minimum intensity: **87**
- Homogeneity ratio: **0.5650**
